Reading and writing network data¶

io.pandas

Functions to read and write network and path data to/from pandas data frames

io.csv

Functions to read and write csv tables.

io.sql

Functions to read and write sql database tables.

io.graphtool

Functions to read networks from graphtool binary format and to retrieve network data from the netzschleuder repository

io.graphml

Functions to read and write graphml files.

io.konect

Functions to read files from the Koblenz Network Collection (konect.cc).

io.infomap

Functions to export paths and higher-order models to state files used by InfoMap

Methods to analyze clustering patterns in networks

statistics.degrees

Methods to calculate degree statistics and assortativity in static networks.

statistics.likelihoods

Methods to calculate model likelihoods based on observed paths.

statistics.modularity

Methods to assess the modular structure of networks

statistics.subpaths

Methods to calculate sub-path statistics in Paths and PathCollections
