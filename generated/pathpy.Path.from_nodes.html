pathpy.Path.from_nodes¶

classmethod Path.from_nodes(nodes: 'Sequence[Node]', **kwargs: 'Any') → 'Path'[source]¶

Generate a Path object from a list of nodes.

Parameters
nodesList[Node]

Nodes from a list of Node objects are added to the path.

kwargsAny, optional(default={})

Attributes assigned to the path as key = value pairs.

Returns
Path

Returns a Path object with nodes and edges according to the given list of Node objects.

Examples

Generate a simple path.

>>> from pathpy import Path
>>> p = Path.from_nodes(['a', 'b'])
>>> p.nodes
{'a': Node a, 'b': Node b}
