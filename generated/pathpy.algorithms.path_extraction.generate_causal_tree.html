pathpy.algorithms.path_extraction.generate_causal_tree¶

pathpy.algorithms.path_extraction.generate_causal_tree(dag, root, node_map)[source]¶

For a directed acyclic graph and a non-injective mapping of nodes, this method creates a causal tree for a given root node. This is useful for the extraction of causal paths in time-unfolded DAG representations of temporal networks. The nodes “{v}_{d}” in the resulting causal tree capture that - starting from the root node at step 0 - there is a causal path to node v at distance d from the root. Note that the same node can be represented by multiple nodes in the causal tree (at different distances d).
