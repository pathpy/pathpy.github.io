pathpy.Edge.uid¶

property Edge.uid¶

Return the unique identifier (uid) of the edge.

Returns
str

Return the edge identifier as a string.

Examples

Generate a single edge and print the id.

>>> from pathpy import Edge
>>> vw = Edge('v','w')
>>> vw.uid
'v-w'
