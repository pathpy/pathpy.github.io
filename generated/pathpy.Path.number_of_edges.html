pathpy.Path.number_of_edges¶

Path.number_of_edges(unique: bool = True) → int[source]¶

Return the number of edges in the path.

Parameters
unique: bool, optional(default=True)
If unique is ``True`` only the number of unique edges in the path
is returnd.
Returns
int
Returns the number of edges in the path.

Examples

Generate a simple path.

>> > from pathy import Path >> > p = Path(‘a’, ‘b’, ‘c’, ‘a’, ‘b’)

Get the number of unique edges:

>> > p.number_of_edges() 3

Get the number of all observed edges in the path:

>> > p.number_of_nodes(unique=False) 4
