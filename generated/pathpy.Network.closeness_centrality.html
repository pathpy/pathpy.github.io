pathpy.Network.closeness_centrality¶

Network.closeness_centrality(normalized: bool = False) → Dict¶

Calculates the closeness centrality of all nodes.

Note

If normalized=False (Default) for each node v the closeness centrality is given as 1/sum_w(dist(v,w)) where dist(v,w) is the shortest path distance between v and w. For normalized=True the counter is multiplied by n-1 where n is the number of nodes in the network. Shortest path distances are calculated using the function shortest_paths.distance_matrix.

Parameters
networkNetwork

The Network object that contains the network

normalizedbool

If True the resulting centralities will be normalized based on the average shortest path length.

Examples

Compute closeness centrality in a simple network

>>> import pathpy as pp
>>> net = pp.Network(directed=False)
>>> net.add_edge('a', 'x')
>>> net.add_edge('x', 'b')
>>> c = pp.algorithms.centralities.closeness_centrality(net)
>>> c['a']
0.3333333333333333
