pathpy.Network.add_node¶

Network.add_node(*node: Union[str, pathpy.core.node.Node], **kwargs: Any) → None[source]¶

Add a single node to the network.

Parameters
nodeNode

The Node object, which will be added to the network.

kwargsAny, optional (default = {})

Attributes assigned to the node as key=value pairs.

Examples

Generate an empty network and add single nodes.

>>> from pathpy import Node, Network
>>> a = Node('a')
>>> net = Network()
>>> net.add_node(a, color='azur')
>>> net.nodes
{'a': Node a}

Generate new node from string uid.

>>> net.add_node('b', color='blue')
>>> net.nodes
{'a': Node a, 'b': Node b}
