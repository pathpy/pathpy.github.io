pathpy.Network.number_of_nodes¶

Network.number_of_nodes(self, unique: 'bool' = True) → 'int'[source]¶

Return the number of nodes in the network.

Parameters
uniquebool, optional (default = True)

If unique is True only the number of unique nodes in the network is returnd.

Returns
int

Returns the number of nodes in the network.

Examples

Generate a simple network.

>>> from pathy import Path, Network
>>> net = Network('a-b-c-a-b')

Get the number of unique nodes:

>>> net.number_of_nodes()
3

Get the number of all observed node visits in the network:

>>> net.number_of_nodes(unique=False)
5
