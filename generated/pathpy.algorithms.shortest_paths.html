pathpy.algorithms.shortest_paths¶

Algorithms for shortest paths calculations.

Functions

all_longest_paths(network[, weight])

Returns a dictionary containing all longest shortest paths, i.e. shortest paths that correspond to the diameter of the network, between all pairs of nodes.

all_shortest_paths(network[, weight, …])

Calculates shortest paths between all pairs of nodes.

avg_path_length(network[, weight, exclude_zero])

Calculates the average shortest path length in directed or undirected

diameter(network[, weight])

Calculates the length of the longest shortest path

distance_matrix(network[, weight])

Calculates shortest path distances between all pairs of nodes

shortest_path_tree(network, source[, weight])

Computes a shortest path tree rooted at the node with the given source uid.

single_source_shortest_paths(network, source)

Calculates all shortest paths from a single given source node using a custom implementation of Dijkstra’s algorithm based on a priority queue.
