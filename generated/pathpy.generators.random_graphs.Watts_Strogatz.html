pathpy.generators.random_graphs.Watts_Strogatz¶

pathpy.generators.random_graphs.Watts_Strogatz(n: int, s: int, p: float = 0.0, loops: bool = False, node_uids: Optional[list] = None) → pathpy.models.network.Network[source]¶

Undirected Watts-Strogatz lattice network

Generates an undirected Watts-Strogatz lattice network with lattice dimensionality one.

Parameters
nint

The number of nodes in the generated network

sfloat

The number of nearest neighbors that will be connected in the ring lattice

pfloat

The rewiring probability

Examples

Generate a Watts-Strogatz network with 100 nodes

>>> import pathpy as pp
>>> small_world = pp.algorithms.random_graphs.Watts_Strogatz(n=100, s=2, p=0.1)
>>> print(small_world.summary())
