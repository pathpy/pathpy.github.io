pathpy.Node.update¶

Node.update(self, other: 'Node' = None, attributes: 'bool' = True, **kwargs: 'Any') → 'None'[source]¶

Update of the node object.

Update the node with new kwargs or based on an other given Node object. If an other object is given, the other attributes can be used or not.

Parameters
otherNode, optional (default = None)

An other Node object, which is used to update the node attributes and properties.

attributesbool, optional (default = True)

If True the attributes from the other node are written to the initial node. If False only the incoming and outgoing edges are updated.

kwargsAny

Keyword arguments stored as node attributes. Attributes are added to the node as key=value pairs.

Examples

Create an Node object with an assigned attribute.

>>> from pathpy import Node
>>> u = Node('u', color='red')
>>> u['color']
'red'

Update color and shape of the node.

>>> u.update(color='green', shape='rectangle')
>>> u['shape']
'rectangle'

Create a new node.

>>> v = Node('v', shape='circle', size=30)
>>> u.update(v)
>>> u.attributes
{'color': 'green', 'shape': 'circle', 'size': 30}
