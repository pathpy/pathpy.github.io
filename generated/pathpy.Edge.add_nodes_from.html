pathpy.Edge.add_nodes_from¶

Edge.add_nodes_from(self, nodes: 'List[Node]', **kwargs: 'Any') → 'None'[source]¶

Add multiple nodes from a list.

Parameters
nodesList[Node]

Nodes from a list of Node objects are added to the edge.

kwargs : Any, optional (default = {})

Attributes assigned to all nodes in the list as key=value pairs.

.. note::

This function is only used internally. It can be used to consider hyperedges in a futher version of pathpy.
